SNES Controller

On 01/31/2018 I was asked to make an instument for my music apperication course. Having little to no background in music, I decided to do what any good computer science student would do. Program my own instument. My professor was very kind enough to allow me to do so. With this assignment I made an SNES controller make different soundwaves, emulating what you would find on an NES. It's nothing a purist would like and with how dirty this code was written, id imagine people screaming that do music at this.

Ill go ahead and explain the notation.

Normal Notes



The meat and potatos of this instument is done mostly with the d-pad of the controller. This uses whatever soundwave is selected by using to change the timbre of the instument. The d-pad along used with the as a modifier makes the sound go up in pitch to reach more than just four notes. Using diagonals will make noises between the two sounds.



Bass Notes





The bass notes on the right side of the controller can be used in combos to make different sounds. For example, if you press and at the same time it will play a note inbetween those two notes. If you use a with any bass notes, it will make the note sound higher. Giving you a good range to work with for a bassline.

Snare

I was only really able to get a snare drum using a bit of whitenoise. If you cut off whitenoise really quickly it makes a reasonably believable snare drum sound. It's enough to make this controller be a one-man-band type of instument.
